a) Hoy en día, Git es, con diferencia, el sistema de control de versiones 
moderno más utilizado del mundo. Git es un proyecto de código abierto 
maduro y con un mantenimiento activo que desarrolló originalmente Linus 
Torvalds, el famoso creador del kernel del sistema operativo Linux, en 
2005.
