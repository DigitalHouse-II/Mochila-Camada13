a. ¿Quién inventó el sistema de control de versión Git y por qué?
Github fue desarrollado por Chris Wanstrath, P. J. Hyett, Tom Preston-Werner y Scott Chacon usando Ruby on Rails, y empezó en 2008. Aunque la compañía, Github, Inc, existía desde 2007. Pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares.
