¿Quién inventó el sistema de control de versión Git y por qué? Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
¿A quién pertenece actualmente Github y por qué? El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares.
